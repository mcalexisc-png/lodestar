# Architecture Decision Records

This directory holds short records of notable technical decisions: what was
decided, why, and what alternatives were considered. They're a log, not a
spec — once written, an ADR is rarely edited; a later decision that changes
course gets its own ADR that references the one it supersedes.

Format: `NNNN-title-with-dashes.md`, numbered sequentially.
